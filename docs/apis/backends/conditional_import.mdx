```python
module: ModuleType = conditional_import(
    name: str
)
```

{<p>Import the module if it exists. Otherwise, return a [`ModulePlaceholder`](./ModulePlaceholder.mdx).</p>}

## Arguments

### Requires

| Argument | Type  |             Required              | <center>Description</center>           |
| :------: | :---: | :-------------------------------: | -------------------------------------- |
|  `name`  | `str` | <InlineIcon icon={vsiCheckIcon}/> | The name of the module to be imported. |

### Returns

| Argument |     Type     | <center>Description</center>                                                                                                |
| :------: | :----------: | :-------------------------------------------------------------------------------------------------------------------------- |
| `module` | `ModuleType` | If the module exists, this value is the imported module. Otherwise, it is a [`ModulePlaceholder`](./ModulePlaceholder.mdx). |
