```python
flag: bool = is_module_invalid(
    module: ModuleType
)
```

Check whether the given module is invalid or not.

An invalid module does not provide any functionalities but only serves as a
placeholder.

## Arguments

### Requires

| Argument |     Type     |             Required              | <center>Description</center> |
| :------: | :----------: | :-------------------------------: | ---------------------------- |
| `module` | `ModuleType` | <InlineIcon icon={vsiCheckIcon}/> | The module to be checked.    |

### Returns

| Argument |  Type  | <center>Description</center>                                                                                                                                                             |
| :------: | :----: | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|  `flag`  | `bool` | If this flag is `True`, the module will be notated as [`ModulePlaceholder`](./ModulePlaceholder.mdx), i.e. the module will be marked as invalid. Otherwise, the module should be usable. |
